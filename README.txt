В файл добавлена Папка TEST_TKINTER, в которой в файле main2.py создан ШАБЛОК приложения, который использует основные инструменты
entry
button
messagebox
label
frame

В файле test.py идет тестирование функций приложения.
При запуске теста через Терминал или Инструменты IDE, будут выведены результаты тестов.

Тесты подставляют данные и проверяют их через библиотеку unittest.
